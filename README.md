NextHire – App Móvil Administrativa
Panel de administración para empresas y gestores de talento

NextHire es una aplicación móvil desarrollada en Android Studio, utilizando una arquitectura híbrida compuesta por XML + Jetpack Compose, y conectada a APIs hospedadas en Azure.
